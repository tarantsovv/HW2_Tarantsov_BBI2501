# Консольное приложение для управления рецептами

Второе домашнее задание по курсу «Технологии программирования». Это небольшая консольная программа на Python, которая помогает работать с рецептами и автоматически собирать списки покупок. 

## Что реализовано в проекте:

* **Класс `Ingredient`** — описывает конкретный продукт (название, сколько его нужно и в чем измеряется). Внутри стоит проверка, чтобы нельзя было передать отрицательное количество.
* **Класс `Recipe`** — собирает ингредиенты в одно блюдо. Умеет увеличивать или уменьшать порции (масштабировать рецепт), а если добавить один и тот же продукт дважды, он просто сложит их количество.
* **Класс `DietaryRecipe`** — наследник обычного рецепта, но с указанием диеты (например, веганский или кето). При выводе в консоль добавляет эту метку в начало.
* **Класс `ShoppingList`** — собирает вместе ингредиенты из разных рецептов, сам суммирует совпадающие продукты и сортирует весь список по алфавиту перед походом в магазин. Еще можно удалять ненужные рецепты или складывать два списка через обычный `+`.
